Notes: Memory location $fe contains a new random byte on every instruction. Memory location $ff contains the ascii code of the last key pressed. Memory locations $6000 to $7fff map to the screen pixels. Different values will draw different colour pixels. The colours are: $0: negro (00aa00#) $1: verde (#00aa00) $2: rojo (#ff00ff) $3: naranja (#ffaa00) $4: botella (#005500) $5: lima (#00ff00) $6: ladrillo (#ff5500) $7: amarillo (#ffff00) $8: azul (#0000ff) $9: celeste (#00aaff) $a: magenta (#ff00ff) $b: rosita (#ffaaff) $c: azur (#0055ff) $d: cian (#00ffff) $e: fucsia (#ff55ff) $f: blanco (#ffffff)
