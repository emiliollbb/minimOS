Notes: Memory location $fe contains a new random byte on every instruction. Memory location $df9a contains the ascii code of the last key pressed. Memory locations $6000 to $7fff map to the screen pixels. Different values will draw different colour pixels. The colours are: $0: Black (#000000) $1: Green (#00aa00) $2: Red (#ff0000) $3: Orange (#ffaa00) $4: Bottle (#005500) $5: Lime (#00ff00) $6: Brick (#ff5500) $7: Yellow (#ffff00) $8: Blue (#0000ff) $9: Sky (#00aaff) $a: Magenta (#ff00ff) $b: Pink (#ffaaff) $c: Azur (#0055ff) $d: Cyan (#00ffff) $e: Fuchsia (#ff55ff) $f: White (#ffffff)
